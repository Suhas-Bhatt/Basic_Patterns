# 📐 Pattern Visualizer

An interactive browser-based tool that renders 9 classic mathematical star patterns — ported directly from the Java console programs in `pattern.java`.

Built with pure vanilla HTML, CSS, and JavaScript. Features a premium dark glassmorphism UI.

---

## ✨ Patterns Available

| Pattern | Description |
|---------|-------------|
| Hollow Rectangle | Rectangular border made of stars |
| Mirrored Half Pyramid | Right-aligned growing pyramid |
| Inverted Half Pyramid (Numbers) | Shrinking numbered rows |
| Floyd's Triangle | Sequential number triangle |
| 0-1 Triangle | Checkerboard binary triangle |
| Butterfly Pattern | Symmetric hourglass wings |
| Solid Rhombus | Filled diagonal rhombus |
| Hollow Rhombus | Hollow diagonal rhombus |
| Diamond Pattern | Symmetric centered diamond |

---

## 📂 Project Structure

- **[index.html](./index.html)** - Pattern Visualizer UI layout
- **[style.css](./style.css)** - Premium dark glassmorphic styling
- **[app.js](./app.js)** - All 9 pattern generation algorithms
- **[pattern.java](./pattern.java)** - Original Java source patterns

---

## 🚀 How to Run

### Web Visualizer
Open **[index.html](./index.html)** in any modern browser. No setup required.

### Java Console Output
```bash
cd Basic_Patterns-main
javac pattern.java
java pattern
```
